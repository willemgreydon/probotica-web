# ProBotica GOD-TIER Scaffold

Ultra-premium Next.js App Router scaffold with cinematic motion, GSAP ScrollTrigger, horizontal pinned scroll, grain/noise texture, cursor interaction, OpenAI API scenario endpoints, strict TypeScript, Tailwind tokens and Vercel-ready env.

## Start
```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

Open `http://localhost:3000`.

## AI Scenarios
- `/api/ai/lead-qualifier`
- `/api/ai/ux-audit`
- `/api/ai/content-studio`

Set `OPENAI_API_KEY` for live AI. Without it, demo fallback stays usable.

## Premium interactions included
- Cinematic editorial hero inspired by the supplied Watson Masters reference
- Global animated grain/noise overlay
- GSAP text/card reveal on scroll
- Sticky horizontal scroll agent showcase
- Custom magnetic-style cursor
- Glass cards, scanline overlays, dynamic gradients
- Reduced-motion fallback
