# Product & Content Logic

## **How Should Prompt Pack, Bot Pack, and Bot Suite Differ Content-Wise?**

### **Prompt Pack**

**What it is:**

- Single, expertly-crafted system prompt
- For a specific use case (e.g., Ad Campaigns, Surveys, Content Ideas)
    
    **Audience:**
    
- Users who want to upgrade their results in AI tools fast
- No code or setup—just copy & paste
    
    **Content Focus:**
    
- Hero: “Unlock instant results with a single prompt.”
- What’s included: 1 prompt, guide, use case examples
- Simple, benefit-focused
- Lots of “Before/After” or sample outputs
- FAQ: “Which AI tools? How do I use it?”
- Perfect for first-time buyers & solo users

---

### **Bot Pack**

**What it is:**

- Set of ready-to-use AI bots (chatbots, automations, templates)
- Usually for one domain (e.g., Marketing, HR, Education)
    
    **Audience:**
    
- Power users, teams, small businesses
- No/low-code, sometimes requires minimal setup
    
    **Content Focus:**
    
- Hero: “Unlock multiple AI bots for your workflow.”
- What’s included: e.g., 3–5 bots, deployment guide, integrations
- Comparison chart: Each bot’s purpose & strengths
- Video demo or walkthrough recommended
- FAQ: “How do I install/use the bots? Is support included?”
- Highlight scalability, time-savings, versatility

---

### **Bot Suite**

**What it is:**

- Complete, all-in-one AI solution (multiple bots + prompts + advanced features, possibly a dashboard)
- For teams, agencies, enterprise—maximum power
    
    **Audience:**
    
- Professional teams, agencies, enterprises
- Looking for full workflow automation & integration
    
    **Content Focus:**
    
- Hero: “Your complete AI-powered workflow suite.”
- What’s included: All bots, premium prompts, user management, analytics, premium support
- Feature highlights: Integration, reporting, team roles
- ROI calculator or case studies/testimonials
- Demo scheduling or onboarding video
- FAQ: “What’s the difference to Bot Pack? Is onboarding/support included? Can we customize?”
- Emphasize ROI, security, team productivity, white-glove support

## 2. **SKU & Category Logic – Best Practice for Shop**

### **Structure**

- **Three main product types:** Prompt Pack, Bot Pack, Bot Suite
- **Categories:** Each solution/field (e.g., Marketing, HR, Education, Sales, Analytics, etc.)
- **10+ packs per solution**

### **SKU Naming Logic (Best Practice):**

Keep it **readable, scalable, and unique** for inventory, analytics, and marketing.

Suggested logic:

`[Type Abbreviation]-[Category]-[PackName/ShortCode]-[Version/Number]`

**Examples:**

- Prompt Pack, Marketing, Ad Optimizer, v1: `PP-MKT-ADOPT-01`
- Bot Pack, HR, Onboarding Bots, v2: `BP-HR-ONB-02`
- Bot Suite, Sales, Enterprise, v1: `BS-SLS-ENT-01`

**Legend:**

- `PP` = Prompt Pack
- `BP` = Bot Pack
- `BS` = Bot Suite
- `MKT` = Marketing, `HR` = Human Resources, etc.
- `ADOPT` = Ad Optimizer, `ONB` = Onboarding, `ENT` = Enterprise, etc.
- Version/Pack number for future upgrades

### **Shop Navigation/Filtering:**

- **Category filter:** Marketing, HR, Education, etc.
- **Type filter:** Prompt Pack / Bot Pack / Bot Suite
- **Use case filter:** (Ad Optimization, Customer Service, Training, etc.)
- **Search by SKU**: Useful for support or B2B clients

---

## **Summary/TL;DR**

- **Prompt Pack** = 1 expert prompt, 1 use case, ultra-simple, solo users
- **Bot Pack** = Set of bots for one workflow/domain, for teams/power users
- **Bot Suite** = Complete solution: all bots, prompts, extras for agencies/enterprise
- **SKU:** [Type]-[Category]-[Pack]-[Number] (e.g., `BP-MKT-ADOPT-01`)
- **Categories:** By solution/domain, so users can browse by field or type
